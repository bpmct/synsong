Binary/English conversion or XOR cipher!



How it works

If you give one input into the first box, it will detect whether you have inputted English or binary and give you the translation from that to the other.

If you give two binary or two English inputs, it will use a XOR cipher to encript your text and return the encrypted output.

If you give a binary input and a English input, it will decript your phrase using a XOR cipher again and return the English output. It's case sensitive but only works with letters!
